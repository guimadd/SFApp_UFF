resetFields()

    Função: Limpa todos os campos de entrada e saída na interface do usuário, esconde o botão de decodificação e mostra o botão de codificação.

Node

    Função: Construtor para criar nós em uma árvore binária de busca (BST). Cada nó contém dados, um bit, ponteiros para os nós esquerdo e direito e a soma das frequências.

show()

    Função: Retorna os dados contidos em um nó específico da árvore.

insert(freq)

    Função: Insere elementos na árvore binária de acordo com as probabilidades calculadas (ver calcSlice()).
    Algoritmo: Insere nós na árvore binária com base na divisão de frequências.

calcSlice(freq)

    Função: Calcula o ponto de divisão do array de frequências para inserção na árvore binária.
    Algoritmo: Divide o array de frequências em duas partes com somas de frequências aproximadas.

createFreqTable(string)

    Função: Cria uma tabela de frequência a partir de uma string.
    Algoritmo: Contagem de caracteres. Itera sobre a string e conta a frequência de cada caractere.

compare(a, b)

    Função: Função de comparação para ordenar os pares de frequências em ordem decrescente.
    Algoritmo: Utilizado para ordenar a tabela de frequência.

generateCodes(node, path, codeDict)

    Função: Gera códigos binários para cada caractere baseado na árvore binária construída.
    Algoritmo: Gera códigos percorrendo a árvore binária.

encode(string)

    Função: Codifica uma string usando os códigos binários gerados.
    Algoritmo: Substitui cada caractere pelo seu código binário correspondente.

decode(bitCode, codesObj)

    Função: Decodifica uma string codificada em binário usando o dicionário de códigos.
    Algoritmo: Percorre o código binário e substitui pelos caracteres originais.

initDecode(bitCode, codesObj)

    Função: Inicializa o estado de decodificação.

decodeStep()

    Função: Realiza a decodificação passo a passo.
    Algoritmo: Percorre o código binário passo a passo e decodifica em tempo real.

updateUI()

    Função: Atualiza a interface do usuário com o estado atual da decodificação.
